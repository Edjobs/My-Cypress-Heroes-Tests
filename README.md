# My Cypress Heroes Tests

This is a project that utilizes cypress to create tests for the "Cypress heroes" app (https://github.com/cypress-io/cypress-heroes/tree/main), in which you need to install it before. 

It tests the app's login, creating, changing and deleting the heroes. 

## Requirements

Need to install Cypress heroes and then copy my (this git's) cypress files into them.

> Note:
>
> The cypress heroes folder is alongside my cypress folder.
>
> In case of **npm run dev** not working, try entering the folder with.
>
> ````
> cd cypress-heroes
> ````
> 
> And then outside of this folder, running cypress.
>

## Location. 

when opening cypress. my test is on the ** 3 - My projects **
